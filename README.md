# Theme Template

###### Requirements:

- node
- npm install gulp-cli -g
- npm init
  - npm install --save-dev gulp gulp-postcss gulp-sass gulp-sourcemaps gulp-uglify gulp-environments autoprefixer cssnano browser-sync jquery popper bootstrap tether panini

## Gulp Tasks

- default
- clean
- js-compile
- compile-js
- compile-bootstrap
- compile-scss
- compile-html
- watch
